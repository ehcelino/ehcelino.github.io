Bem vindo ao meu portfólio

Meu nome é Eduardo, e minha ambição é me tornar um desenvolvedor. Neste site você encontrará alguns exemplos de meu trabalho até agora, bem como informações sobre minha formação e um pouco a meu respeito.

Minhas habilidades

Python

Aprendi a programar em Python por conta própria, com o intuito de criar um software para gerenciar o Estúdio Lótus, um espaço de treinamento e condicionamento físico multidisciplinar que pertence a minha companheira. Para isso assisti a vários vídeos de introdução e treinamento, pesquisei vários artigos e tutoriais, e fiz uso abundante da documentação da linguagem, finalmente tendo êxito em criar um software personalizado e funcional. Tenho domínio básico da linguagem, sou capaz de utilizar bibliotecas para criação de interface gráfica, manusear bancos de dados MySql, manipular o sistema de arquivos e trabalhar com classes e objetos. Escrevi também outros pequenos programas na linguagem que disponibilizei através do meu github. Para maiores informações e detalhes sobre estes projetos, clique aqui.

Ruby e Ruby on Rails

Aprendi a programar em Ruby participando do programa "Quero Ser Dev 40+" da CampusCode.com.br, patrocinado pela Locaweb Company. Durante o treinamento, desenvolvi código para resolver os exercícios propostos, iniciando por lógica de programação em Ruby, e depois princípios de Ruby on Rails. Depois do treinamento continuei meus estudos através do site The Odin Project, e desenvolvi alguns projetos para aprimorar meus conhecimentos, dos quais o mais avançado é uma loja online, com controle de produtos, carrinho de compras, entre outros. Este projeto se encontra em meu github, e também pode ser acessado no Render aqui. Para maiores detalhes sobre este projeto, clique aqui.
